# sdev245-mod3

In this module I have created three separate files.

In SHA-256 I have an app that uses the hashlib library to take an input string and print a hashed message. There is also a function to take an input file name and return a hash of the data within the file

In Caeser Cipher I have an app that will take a message as input. It then will return the encoded message as well as the decoded message.

In Digital Signature I have an app that creates digital signatures with the rsa library. It then verifies the message is accurate and untampered.
